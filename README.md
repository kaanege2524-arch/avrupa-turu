# ✈️ Avrupa Turu 2026
Avrupa'nın en güzel şehirlerini keşfetmek için tasarlanan modern React uygulaması.

## 🚀 Kurulum
```bash
npm install
```

## 📝 Geliştirme Modu
```bash
npm run dev
```
Uygulama `http://localhost:5173` adresinde açılır.

## 🏗️ Production İçin Derleme
```bash
npm run build
```

## 👀 Build'i Önizlemek
```bash
npm run preview
```
